# AWS - Short Introduction 

Amazon Web Services (AWS) is a leading cloud computing platform that offers a broad range of services, such as computing, storage, databases, analytics, and machine learning,  deployed on a highly scalable and reliable infrastructure. AWS provides developers and organizations with the flexibility to utilize its global network of data centers, enabling seamless deployment and management of applications while optimizing costs with a pay-as-you-go pricing model. Its comprehensive suite of services caters to diverse computing needs, making AWS a prominent choice in the cloud computing landscape.

## Overview
- Create an account on the AWS platform
- Add a user to create an access key and secret key on AWS
- Log in to Deploypro
- Add your AWS keys to your Deploypro Account
- Deploy a static app in Deploypro
